# HR-Analytics Bot — HR Insight

**Что это:** HR-Analytics Bot — персональный AI‑ассистент для HR, который автоматически анализирует данные сотрудников (Excel/CSV), вычисляет ключевые метрики и выдаёт понятные рекомендации. Проект фокусируется на быстрой интеграции в HR‑процессы и наглядной подаче выводов (KPI‑карточки, графики, текстовые рекомендации).

**Ценность:**
- Экономит время HR‑команд, убирая рутину при подготовке отчётов.
- Помогает быстрее находить проблемные отделы и возрастные группы с высоким оттоком.
- Дает практичные рекомендации и план найма на основе реальных данных.
- Работает локально (без передачи данных), что важно для конфиденциальности персонала.

**Ключевые функции:**
- Загрузка и автоматическая очистка Excel/CSV (удаление строки типов).
- Готовые отчёты: текучесть, отток по возрасту, тренд headcount, распределение tenure, план найма.
- Интерпретация результатов на естественном языке и экспорт отчётов.
- Безопасная песочница для запросов (read‑only), опция интеграции с LLM для расширенной аналитики.

**Как использовать (вкратце):**
1. Загрузить файл с данными (Excel/CSV).
2. Выбрать готовый отчёт или задать вопрос в чат‑интерфейсе.
3. Получить графики, цифры и краткую рекомендацию.

**Контакты:** в репозитории — инструкции по запуску, примеры тестовых запросов и демо‑данные.
